# Rick and Morty Episode Fetcher

This set of files demonstrates how to fetch specific episode data from the Rick and Morty GraphQL API.

## Query Details
- **Field**: `episode(id: ID!)`
- **Arguments**: `id` (The unique identifier of the episode)
- **Requested Fields**: `id`, `name`, `air_date`, `episode`

## Data Source
Endpoint: https://rickandmortyapi.com/graphql
